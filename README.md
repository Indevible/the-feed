# The Feed

**The market may notice XDEL. Indelible notices what people do with it.**

The Feed is the public activity layer being developed for Indelible and XDEL.

Its purpose is to interpret specially structured XDEL activity as a lightweight social language. Wallet transfers, swaps, and trades may carry appended numeric expressions that The Feed can detect, translate, and present as entertaining, shareable public events.

The amount still carries economic value. Appended digits may carry expression.

## Current status

**Early public design**

The language, parsing rules, terminology, event classes, visual treatments, and community integrations are being defined in public. No production protocol is represented as complete.

## Principles

- Price is not the language.
- Numeric patterns can carry meaning independently of scale.
- Interaction matters more than speculation.
- The Feed should be entertaining without becoming a negative reputation ledger.
- Interpretation of public activity is not proof of private intent.
- Durable Vouch mechanics are developed separately and are outside this repository.

## Repository boundary

This repository is limited to event observation, numeric-language interpretation, normalized Feed events, visual presentation, and community integrations.

It does not contain wallet keys, transaction signing, custody logic, treasury operations, Vouch permanence mechanics, or governance over personal standing.

## First milestone

The first milestone is a written transaction grammar, a normalized Feed event model, and a small provisional lexicon tested with offline examples before live indexing begins.
